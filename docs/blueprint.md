# **App Name**: RailTracer

## Core Features:

- QR Code Scanning: Scan laser-engraved QR tags on railway assets using the device camera to quickly identify components.
- Component History Viewer: Display the complete history of a component, including past inspections, repairs, and replacements.
- Inspection Record Updater: Authorized staff can update inspection records with new findings, notes, and status changes. Generative AI tool will offer suggested next actions based on defect severity and repair history.
- Role-Based Content: Display different content and functionalities based on the user role (e.g., Authorized Staff vs. Public). Public should only know details and staff should have login credentials to access all full control.
- Offline Sync Indicator: Provide a clear visual indicator of the app's offline sync status, showing when data is synchronized with the server.
- Data-Rich Dashboard: Display a data-rich dashboard for admin users, showing key metrics, component states (Verified, Unverified, Damaged), and analytics.
- Component state management: Components shall be tagged with a state and each state will have clear definition and implications
- AI Chatbot: Provide an AI-powered chatbot for users to ask questions about components, inspections, and procedures. The chatbot will function as a tool, using reasoning to answer questions.
- AI-assisted Material Status Detection: If the material status cannot be automatically detected, allow the user to show the component to the AI. The AI will then analyze the component via the camera and provide information about its condition, such as whether it's corrosive or not. The AI will function as a tool, using reasoning to provide the user with the appropriate information.

## Style Guidelines:

- Primary color: Dark steel blue (#34495E) to convey trustworthiness and industrial feel.
- Background color: Light grey (#ECF0F1) for a clean, efficient backdrop.
- Accent color: Vivid orange (#E67E22) to highlight CTAs and important status indicators.
- Body and headline font: 'PT Sans', a humanist sans-serif for modern, legible, readable text.
- Use a set of consistent, minimalist line icons to represent different component types, inspection actions, and states.
- Modern, clean layout with a focus on clear visual hierarchy. Utilize card-based design for components and inspection records.
- Subtle animations for transitions and data updates to provide smooth feedback to the user.
